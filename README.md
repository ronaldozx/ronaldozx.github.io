# 🚀 Portfolio Profissional - Guia Completo

Portfolio pessoal desenvolvido com React, TypeScript e Styled Components. Design moderno, animações suaves e totalmente responsivo.

## ✨ Características Principais

- 🎨 **Design Moderno** - Interface elegante com efeitos glassmorphism e gradientes
- 🌌 **Background Animado** - Galáxia com estrelas piscantes e nebulosas
- 🎭 **Animações Suaves** - Transições fluidas em todos os elementos
- 📱 **Totalmente Responsivo** - Perfeito em desktop, tablet e mobile
- ⚡ **Alta Performance** - Otimizado para carregamento rápido
- ♿ **Acessível** - ARIA labels e navegação por teclado

## 📸 Seções do Portfolio

### 🏠 Hero - Primeira Impressão
- Apresentação com animações de entrada
- Links para redes sociais
- Botão de download do currículo

### 👤 Sobre Mim
- Informações pessoais
- Estatísticas (experiência, projetos, clientes)

### 💪 Habilidades
- 4 categorias de skills (Frontend, Backend, Ferramentas, Design)
- Barras de progresso animadas

### 🚀 Projetos
- Grid responsivo de projetos
- Cards com imagens e descrições

### 📧 Contato
- Formulário funcional
- Informações de contato
- Links para redes sociais
- Animações de interação

## 🚀 Como Usar

### Instalação
```bash
npm install
npm start
```

### Personalização Rápida

1. **Informações Pessoais** - Edite `src/constants/personalInfo.ts`
2. **Habilidades** - Edite `src/sections/Skills/Skills.tsx`
3. **Projetos** - Edite `src/sections/Projects/Projects.tsx`
4. **Cores** - Edite `src/styles/theme.ts`
5. **Adicione suas imagens** em `public/` (eu.png, about.jpg, curriculo.pdf)
